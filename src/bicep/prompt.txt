Generate an Azure deployment resource graph based on user's goal. 


Use the following syntax:

// Group node
{
  id: "<groupId>",
  type: "group"
  hasChildren: true,
}

// Item node
{
  id: "<itemId>",
  type: "<itemType>",
}

// Item node inside a group
{
  id: "<groupId>::<itemId>",
  type: "<itemType>",
}

// Edge: meaning targetNode depends on sourceNode
{
  sourceId: <sourceNodeId>,
  targetId: <targetNodeId>
}
